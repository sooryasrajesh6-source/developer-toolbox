# 🛠️ Developer Toolbox

A responsive all-in-one developer toolkit built using **HTML, CSS, and JavaScript**. This project combines multiple useful utilities into a single, clean interface to improve productivity and demonstrate frontend development skills.

## ✨ Features

- 🔐 Password Generator
- 📱 QR Code Generator
- 📄 JSON Formatter
- 🔤 Text Case Converter
- 📝 Word & Character Counter
- 🎨 Random Color Generator
- 🌈 Gradient Generator
- 🔄 Base64 Encoder / Decoder
- 📏 Unit Converter (KM → Miles)
- 🧮 Calculator
- 🕒 Digital Clock
- ⏱️ Stopwatch
- 📒 Notes App (Local Storage)
- 🌙 Dark / Light Mode
- 📱 Fully Responsive Design

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
